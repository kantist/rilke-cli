## Development Setup

### Prerequisites

-   Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up a Project

Install the Rilke CLI globally:

```
npm install -g @kantist/rilke-cli
```

Create workspace:

```
rilke new [PROJECT NAME]
```

Run the application:

```
cd [PROJECT NAME]
rilke serve
```
